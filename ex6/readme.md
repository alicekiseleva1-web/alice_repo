# Программа для работы с базой почтовых индексов

## Описание
Программа позволяет работать с базой почтовых индексов США

## Возможности
- нахождение местоположения по почтовому индексу
- нахождение индексов по городу и штату
- вычисление расстояния между двумя точками, заданными их почтовыми индексами

## Файлы проекта
- ex6.py — основная программа
- zip_util.py — вспомогательный модуль
- zip_codes_states.csv — файл с данными почтовых индексов

## Запуск
Убедитесь, что все файлы из репозитория находятся в одной папке

```
python ex6.py
```

## Поддерживаемые команды

- loc - позволяет пользователю ввести почтовый индекс и получить город, штат, графство и координаты

пример:
Command ('loc', 'zip', 'dist', 'end') => loc
Enter a ZIP Code to lookup => 12180


- zip - позволяет пользователю ввести город и штат и получить список почтовых индексов

пример:
Command ('loc', 'zip', 'dist', 'end') => zip
Enter a city name to lookup => troy
Enter the state name to lookup => ny


- dist - позволяет ввести два почтовых индекса и получить расстояние между ними

пример:
Command ('loc', 'zip', 'dist', 'end') => dist
Enter the first ZIP Code => 19465
Enter the second ZIP Code => 12180


- end - завершает программу

пример:
Command ('loc', 'zip', 'dist', 'end') => end
Done
